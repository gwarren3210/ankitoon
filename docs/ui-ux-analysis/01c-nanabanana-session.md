## PROMPT 1: 또리 Character Sheet

**Purpose:** Main character design for the tiger mascot

```
Character design sheet for "Ttori" (또리), a cute tiger mascot for a Korean language learning app called Toonky.

STYLE:
- Modern manhwa/webtoon art style
- Chibi proportions (large head, small body, about 2:1 ratio)
- Clean linework, flat colors with minimal shading
- Friendly and approachable, not childish

CHARACTER DETAILS:
- Cute tiger with round face
- Large expressive eyes (dark, shiny)
- Small triangular ears with pink inner
- Orange/amber fur with navy blue stripes
- Fluffy cheeks and small body
- Subtle Korean element: tiny cloud pattern (구름) on forehead OR small traditional collar detail
- Short fluffy tail with stripes

SHOW ON SHEET:
- Front view (default friendly pose, slight smile)
- 3/4 view (walking or waving)
- Side view (profile)
- Back view (showing tail)
- Size reference (shown at different scales: large, medium, small icon)

COLORS:
- Main fur: warm orange (#f97316)
- Stripes: navy blue (#1a1a2e)
- Accent details: electric blue (#4a90d9)
- Eyes: dark with white highlights
- Inner ears/nose: soft pink

BACKGROUND: Light gray or transparent, arranged neatly in grid
FORMAT: Square composition, high resolution
```

## PROMPT 2: 또리 Expression Sheet

**Purpose:** 6 emotions for use across the app

```
Expression sheet for Ttori (또리), a cute chibi tiger mascot. Show the same character from the character sheet in 6 different emotional states.

ARRANGE IN 2x3 GRID:

Row 1:
1. HAPPY (Default) - Friendly smile, eyes open and bright, welcoming pose, maybe small sparkle
2. STUDYING - Determined focused look, slight concentrated frown, maybe holding tiny book or pencil
3. CONFUSED - Head tilted to side, question mark floating nearby, puzzled but cute expression

Row 2:
4. CELEBRATING - Jumping with joy, arms/paws up, eyes closed happy, sparkles and small confetti around, big smile
5. SORRY/APOLOGETIC - Korean-style slight bow, embarrassed sweat drop, apologetic expression, paws together
6. ENCOURAGING - Thumbs up (or paw up), supportive wink or smile, "you can do it" energy, small star nearby

STYLE:
- Consistent with character sheet (manhwa style, chibi proportions)
- Each expression clearly readable at 64px size
- Clean linework, flat colors
- Small decorative elements allowed (sparkles, sweat drop, question mark)

SPECIAL ELEMENT:
- In the CELEBRATING pose, include small Korean cloud motif (구름) floating nearby

COLORS: Same as character sheet (orange fur, navy stripes, blue accents)
BACKGROUND: Light gray or white, expressions clearly separated
FORMAT: 2x3 grid, each cell roughly equal size
```
